# GoodbyeYou

GoodbyeYou is a modular multi-tenant SaaS platform with a multi-page indexed site structure. It is built around state-specific Planning for alternative end-of-life arrangements and includes a built-in Vault for final wishes, last-word preservation, related storage, and access-support information.

The platform is designed for two core user types:

- people planning ahead for their own future arrangements
- people planning now for someone who has died

## Core Platform Areas

- state-specific Planning for alternative arrangement options, guidance, logistics, providers, resources, and official direction
- Vault support for final wishes, last words, stored files, video, access notes, and wet-ink original location tracking

## Indexed Page Structure

- Home
- Plan Ahead
- Planning Now
- Planning
- Alternative Options
- Resources
- Providers
- Vault
- Legal and Official Resources
- About / How It Works

## Local Development

```bash
npm install
npm start
```

## Production Build

```bash
npm run build
```

The build output is generated in `build/`, and the repository root static files are also refreshed to match the current production build.
