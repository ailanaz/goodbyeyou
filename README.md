# GoodbyeYou

Alternative funeral planning resources with state-specific options, providers, and next steps for non-traditional after-death services.

## Pages

- `/` - Home
- `/funeraloptions` - Planning (state search, planning paths)
- `/afterdeathguide` - The Guide (alternative service types)
- `/states/[state]` - State-specific planning pages
- `/resources` - After-death resources
- `/providers` - Provider directory
- `/legal-resources` - Legal and official resources

## Development

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Deployment

GitHub Actions deploys to GitHub Pages on push to `main`. Custom domain: goodbyeyou.com
