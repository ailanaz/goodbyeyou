# GoodbyeYou

Alternative funeral planning resources with state-specific options, providers, and next steps for non-traditional after-death services.

## Pages

- `/` - Home
- `/funeralplanning` - When You Have Time (pre-planning with state search)
- `/when-time-has-run-out` - When Time Has Run Out (immediate-need steps with state search)
- `/states/[state]` - State-specific planning pages
- `/after-death-steps` - After Death Steps

## Development

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Deployment

GitHub Actions deploys to GitHub Pages on push to `main`. Custom domain: goodbyeyou.com
